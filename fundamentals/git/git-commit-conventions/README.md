# Git Commit Message 规范说明

## 1. Commit 类型说明

| type     | 用途                       | 示例                                    |
| -------- | -------------------------- | --------------------------------------- |
| `docs`   | 笔记、README、论文阅读记录 | `docs: add operating systems notes`     |
| `notes`  | 学习笔记内容更新（推荐）   | `notes: update algorithms notes`        |
| `paper`  | 新增或更新论文阅读         | `paper: add notes on Raft paper`        |
| `struct` | 调整目录结构、重命名       | `struct: reorganize notes structure`    |
| `fix`    | 修正错误（笔记勘误）       | `fix: correct explanation of paging`    |
| `chore`  | 杂项（格式、PDF导出等）    | `chore: export algorithms notes to pdf` |
